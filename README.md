# Omni-IDE

Omni-IDE is a powerful, agent-first AI code editor built on the robust core of VS Code. It is designed to bridge the gap between human creativity and AI execution, providing a seamless development experience with integrated AI agents.

## Features

- **Agentic AI Integration**: Built-in support for Omni-Agent to help with coding, research, and project management.
- **Customizable Interface**: A premium, branded experience with custom icons and themes.
- **Extensions Support**: Full compatibility with the Open VSX Registry for your favorite extensions.
- **High Performance**: Leveraging the optimized VS Code runtime for speed and stability.

## Getting Started

### Installation
You can download the latest Windows executable from the [Releases](https://github.com/nihannihu/omni-ide/releases) page.

### Building from Source
1. Clone the repository:
   ```bash
   git clone https://github.com/nihannihu/omni-ide.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development environment:
   ```bash
   npm run watch
   ```
