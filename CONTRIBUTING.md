# Contributing to Omni-IDE

Thank you for your interest in contributing! Omni-IDE aims to be the most advanced agent-connected IDE in the world, and we value your help in achieving that goal.

## How to Get Involved

### Reporting Issues
- Use GitHub Issues to report bugs or suggest features.
- Provide as much detail as possible, including screenshots and environment info.

### Development Workflow
1. Fork the repository.
2. Create a feature branch.
3. Ensure your code follows the existing style patterns.
4. Submit a Pull Request with a clear description of your changes.

### Building the Project
Refer to the [README.md](README.md) for build instructions.

## Code of Conduct
Please be respectful and professional in all interactions within the community.

## Thank You
Every contribution, no matter how small, helps make Omni-IDE better!
